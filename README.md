# Ten Random Points — Animint2 Demo

This repository contains a simple interactive data visualization created using the **animint2** R package.  
It is intended as an **Easy Test submission** for the RStats GSOC 2026 animint2 project.

---

## About the Visualization

- Displays **10 randomly generated points**
- Points are colored based on a simple grouping:
  - `Left` (x < 5)
  - `Right` (x ≥ 5)
- Built using:
  - `ggplot2` for plotting
  - `animint2` for interactive web rendering

The visualization is rendered as a static website and hosted using **GitHub Pages**.
